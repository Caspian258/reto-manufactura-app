# Nexo

Plataforma de organización para equipos universitarios de ingeniería.

## Funcionalidades

- Gestión de equipos con códigos de invitación
- Tareas con prioridad, responsable y fecha límite
- Tablero Kanban con drag-and-drop
- Diagrama Gantt dinámico
- Calendario mensual de entregas
- Comentarios en tareas
- Dashboard global con resumen de actividad

## Stack

- Next.js 16 (App Router)
- Firebase (Auth + Firestore)
- Tailwind CSS v4
- @dnd-kit para drag-and-drop

## Desarrollo local

1. Clona el repo
2. Copia `.env.example` a `.env.local` y completa las variables de Firebase
3. `npm install`
4. `npm run dev`
